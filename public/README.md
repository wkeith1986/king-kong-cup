# Static assets

Drop your King Kong Cup logo image here as **`logo.png`** (or `logo.jpg`).

The site is coded to look for `/logo.png`. If the file doesn't exist, a
typographic "KKC" monogram is shown as a fallback — perfectly fine for
launch. Replace it whenever you have the artwork.

Recommended logo: a square (1:1) PNG with transparent background,
at least 512×512 pixels.
